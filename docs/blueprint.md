# **App Name**: SAS Driver

## Core Features:

- Route Display: Display assigned delivery route with turn-by-turn navigation integration.
- Delivery Confirmation: Enable drivers to confirm successful delivery with timestamp and optional photo upload.
- Status Updates: Allow drivers to update their current status (e.g., 'En Route', 'Delivered', 'Issue').
- Communication Tool: Enable direct communication with dispatch via in-app messaging.

## Style Guidelines:

- Primary color: Light Blue (#6699CC).
- Accent color: Sea Green (#73C2BE).
- Background color: Off-White (#F5F7FA).
- Success color: Forest Green (#2E7D32), Warning color: Amber (#F5A524), Error color: Deep Red (#D23F31).
- Body and headline font: 'Roboto', a clean, readable, medium contrast sans-serif. Note: currently only Google Fonts are supported.
- Emulate a phone viewport (width ~390, height ~844) with a rounded black device frame and safe-area insets.
